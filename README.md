移动端 AI 语音助手 Agent

本项目是一个面向移动端场景的 AI 语音助手 Agent 原型，主要用于验证大模型在手机语音交互、自然语言理解、任务拆解和工具调用中的应用能力。项目通过移动端采集用户语音输入，将语音内容转换为文本后发送至后端 Agent 服务，由大模型完成意图识别、任务判断、步骤拆解和结果生成，最终将处理结果返回手机端，并通过文字展示或 TTS 语音播报反馈给用户。

项目的核心目标是解决传统语音助手只能执行固定关键词指令、无法理解复杂自然语言需求的问题。相比普通语音助手，本项目引入了大模型推理能力和 Agent 工作流，使助手能够更自然地理解用户表达，并根据不同任务类型选择合适的处理方式。例如，普通问答可以直接生成回复；复杂任务会先拆解成多个步骤；涉及外部能力的需求可以调用 API、本地功能或数据接口；连续对话则会结合上下文进行多轮处理。

项目价值
本项目验证了大模型接入移动端语音助手场景的可行性，展示了 AI Agent 在语音交互、任务理解、任务拆解和工具调用方面的能力。它不仅可以作为传统语音助手的智能化升级方案，也可以继续扩展为个人助理、学习助手、办公助手或移动端自动化执行入口。
后续项目可进一步接入系统级能力、个人知识库、日程管理、应用操作接口和多 Agent 协作机制，让语音助手从“回答问题”逐步升级为“理解任务并执行任务”的智能助手。
